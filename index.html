CSS Units

CSS (Cascading Style Sheets) provides a variety of units for specifying measurements and values. Here are the commonly used CSS units.

Absolute Units

px (pixels)

A pixel is a single dot on a computer screen. It is a fixed-size unit that represents the smallest addressable element of a display device.

100px

Relative Units

vw (viewport width):

Represents a percentage of the viewport's width.For instance, 25vw would be a quarter of the width of the viewport.

25vw

vh (viewport height):

Represents a percentage of the viewport's height.For example, 30vh would be half the height of the viewport.

30vh

% (percentage)

A percentage unit represents a proportion of another value. For example, setting a width of 50% on an element means it will take up half the width of its parent container.

50%

em

The "em" unit is relative to the font-size of the parent element. For example, if an element has a font-size of 16 pixels, setting its width to 2em would make it 32 pixels wide.

2em

rem

Similar to the "em" unit, but relative to the root element's font-size. It allows you to create more predictable and consistent sizing across your document.

2em
